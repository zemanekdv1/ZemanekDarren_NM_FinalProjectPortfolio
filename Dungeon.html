Project Dungeon Bosses

This is the very first game that I have worked on, back in 2022. My role for this project was the level and environment designer. Since I was completely new to the concepts and engine we were using (Godot), I chose to add intricacy in the level design itself by adding branching paths and different puzzles/mechanics depending on which route(s) the player took. Below is an in-browser playable version, and also a link to the game's Itch.io page.

Project Dungeon Bosses - Itch.io

Do NOT use the "quit" buttons, they are for the windows executable build. When clicked, they will kill the game. Just refresh the page if you do use it by mistake.
